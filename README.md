# City-nursing-home
A Java GUI app automating hospital transactions with role-based access for Cashier, Manager, and Doctor. Cashiers handle patients and appointments; Managers manage users, patients, doctors, and appointments; Doctors view schedules and patient lists. Data stored in text files for persistence.
